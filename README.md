# YouTube Downloader with GitHub Actions

Automatically download YouTube videos and search results using GitHub's servers.

## How to Use

### 1. Edit commands.txt

Add your commands to `commands.txt`:

```txt
# Search for videos
search artificial intelligence tutorial

# Download a video
download https://youtube.com/watch?v=VIDEO_ID

# Get recent videos from a channel
channel https://youtube.com/@ChannelName
